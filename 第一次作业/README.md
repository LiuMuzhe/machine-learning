# 第一次作业：鸢尾花种类识别

## 项目简介

本项目实现了鸢尾花种类的识别任务，使用了机器学习算法对鸢尾花数据集进行分类。

## 项目结构

- `iris.py` - 主要的Python实现代码
- `iris.ipynb` - Jupyter Notebook版本的实现
- `environment.txt` - 依赖环境配置
- `典型案例学习报告——鸢尾花种类识别.docx` - 详细的学习报告

## 环境配置

1. 确保已安装Python 3.12或更高版本
2. 安装所需的依赖包：

```bash
pip install -r environment.txt
```

## 代码运行步骤

### 方法一：运行Python脚本

1. 打开命令行终端
2. 进入项目目录：

```bash
cd ./第一次作业
```

3. 运行Python脚本：

```bash
python iris.py
```

### 方法二：运行Jupyter Notebook

1. 打开命令行终端
2. 进入项目目录：

```bash
cd ./第一次作业
```

3. 启动Jupyter Notebook：

```bash
jupyter notebook
```

4. 在浏览器中打开生成的URL，点击 `iris.ipynb` 文件
5. 运行所有单元格查看结果

## 结果说明

- 会生成分类结果的可视化图表
- 详细的分析结果请参考学习报告文档
